数据指标说明
新增用户 新增加的应用使用者（按设备），重安装应用不会重复计量。
活跃用户 当日有使用应用（至少启动一次）的用户数。
新用户占比 活跃用户中包含了新增用户和积累下的老用户，此比例为新增用户在其中所占的比率。
启动次数 当日，应用被开启的次数。
人均启动 当日，用户平均使用应用多少次。
平均使用时长 用户平均单次使用应用多长时间。
日活跃率 当日的活跃用户占累计用户比例。
累计用户总数 使用TalkingData以来统计到的新增用户量总值。
一次性用户(%) 自新增日后再没有使用过应用的用户和他们占累计用户的比例。
启动总数 使用TalkingData以来应用被开启的总次数。
每日人均启动 用户在一日中平均使用应用多少次。
周活跃(%) 截至当日，最近一周（含当日的7天）活跃用户数和这些用户占累计用户的比例。
月活跃(%) 截至当日，最近一月（含当日的30天）活跃用户数和这些用户占累计用户的比例。
次日留存率均值 次日留存率是指某日的新增用户中，第二日有使用过应用的用户比例。次日留存率均值是指昨日之前7日每日新增用户的次日留存率的均值。
